# 🌿 GreenNursery - Professional Plant & Tree Nursery App

> A stunning, professional plant nursery application built with React, TypeScript, Tailwind CSS, Three.js, and Framer Motion.

## 🎯 What's Built

### 🖥️ **Complete App Features**
- **Dark Forest Theme** with professional design system
- **3D Interactive Plant Models** using React Three Fiber
- **Smooth Animations** with Framer Motion
- **Responsive Design** for all devices
- **SEO Optimized** with proper meta tags
- **Modern UI Components** using Shadcn/ui

### 📱 **Sections Included**
1. **Navigation Header** - Responsive with mobile menu
2. **Hero Section** - 3D floating plant animation
3. **Plant Categories** - Indoor, Outdoor, Succulents, Flowering
4. **Featured Plants** - Product cards with ratings & pricing  
5. **Interactive 3D Showcase** - Fully interactive 3D plant viewer
6. **Plant Care Guide** - Professional care tips & troubleshooting
7. **Footer** - Newsletter signup & social links

### 🎨 **Design System**
- **Colors**: Dark theme with plant-inspired greens
- **Typography**: Clean, modern fonts with proper hierarchy
- **Animations**: Floating, glowing, and interactive effects
- **Components**: Custom variants with hover states

## 🚀 Quick Start

### 1. **Local Development**
```bash
# Clone & Install
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install

# Start development server
npm run dev
# Open http://localhost:8080
```

### 2. **Build for Production**
```bash
npm run build
```

### 3. **Deploy to Vercel** (Recommended)
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```

**Or use Vercel's GitHub integration:**
1. Push code to GitHub
2. Visit [vercel.com/new](https://vercel.com/new) 
3. Import repository & deploy
